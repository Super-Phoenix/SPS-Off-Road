# Appendix A — Cost & Procurement (summary)

Headline: net-new recurring spend to launch Phase 1 ≈ $0.

- Already in SPS stack (verify tiers, no new accounts): Supabase, Inngest,
  SPS hosting/domain, Solana wallet adapter, Claude Code, repo host.
- Net-new but free (MIT): Phaser 3, Colyseus framework, TS/Vite/React, Node
  headless sim.
- Net-new paid (phased): Colyseus Cloud ~$15/mo OR self-host VPS $5–20/mo
  (Phase 2 only); Supabase tier bump ~$25/mo if needed under tournament load;
  optional transactional email + analytics; Apple/Google dev accounts ONLY if
  Capacitor path is explicitly chosen (not recommended at launch).
- The one material cost: art. Tier 1 programmatic (GLTF→sprites, ~$0) →
  Tier 2 contracted polish ($500–2.5K, triggers Capital Allocation memo) →
  Tier 3 full pass ($2.5–8K, post-traction only). Launch at Tier 1.
- Worst case through Phase 3: ~$45/mo recurring + $0–2.5K discretionary art.
- Pre-launch verification: Supabase limits vs tournament concurrency, Inngest
  headroom, hosting bandwidth/CDN, /race route TLS.

Constraint on this build is founder time, not capital.
